<RichText id="rt-scope" title="Scope">

This local recap covers the local-files privacy mode added for `/visual-plan` and
`/visual-recap`: local MDX authoring, local preview, local Plan app read-only
viewing, and the documentation/skill updates that make the mode discoverable.
The unrelated OAuth-token encryption files currently present in the working tree
are intentionally outside this recap.

</RichText>

The action is `readOnly`, public-agent exposed, and guarded by
`isLocalPlanRuntime()`. It builds a synthetic `PlanBundle` from local MDX files,
sets access to viewer/private, and returns `/local-plans/<slug>` as the local
view path.

export function localPlanFolderName(title: string): string {
  const slug = title
    .normalize("NFKD")
    .replace(/[\\u0300-\\u036f]/g, "")
    .toLowerCase()
    .replace(/[^a-z0-9]+/g, "-")
    .replace(/^-+|-+$/g, "")
    .slice(0, 64)
    .replace(/-+$/g, "");
  return slug || "untitled-plan";
}
